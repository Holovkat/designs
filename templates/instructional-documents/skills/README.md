# Project skills

These skills support the deterministic worktree session lifecycle for a project.

- `worktree-toolkit-init` audits and updates the project-specific session tooling
- `worktree-session-lifecycle` runs the operational session flow on top of the backend shell scripts
- `plan-review` checks planning artifacts against the Q&A/intake record before build approval
- `release-assess` infers release intent/confidence and routes active vectors to specialists
