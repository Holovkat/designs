# Compliance References

`au-standards.md` summarizes the Australian regulations that influence launch scope, including driver fatigue (NHVR Standard Hours), Dangerous Goods (ADG Code), GHS labelling, load restraint requirements, and GS1 identification standards.

## Practical Tie-Ins
- Driver fatigue profiles, credentials, and scheduling validators reference NHVR guidance.
- DG metadata (class, UN number, packing group) and sea handoff gating follow ADG + customs requirements.
- Load restraint and GS1 scanning rules inform the Orders Lifecycle and driver app specs.

The document also tracks roadmap items (hazmat routing, AFM/BFM profiles, compliance portals) so regulatory gaps remain visible.
