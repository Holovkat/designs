# 02 - Folder Layout

This document defines the standard folder structure for the project.

## Root Directory

- **/src:** Contains all the source code.
- **/tests:** Contains all the tests.
- **/docs:** Contains project documentation.
- **/scripts:** Contains build and utility scripts.
- **/.github:** Contains GitHub-specific files like workflows.
- **.env.example:** Example environment file.
- **package.json:** Project dependencies and scripts.
- **README.md:** Project overview.

## /src Directory

- **/src/api:** API-related files (e.g., controllers, routes).
- **/src/config:** Configuration files.
- **/src/core:** Core business logic.
- **/src/lib:** Shared libraries and utilities.
- **/src/models:** Data models and schemas.
- **/src/services:** Services that interact with external APIs or databases.
- **/src/views:** UI components or templates.
